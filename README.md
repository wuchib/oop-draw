# OOP Draw

Monorepo scaffold for a TypeScript-only whiteboard application with a shared Vue renderer for Web and Electron.

## Workspace commands
- `pnpm install`
- `pnpm dev:web`
- `pnpm dev:desktop`
- `pnpm build:web`
- `pnpm build:desktop`
- `pnpm typecheck`
- `pnpm lint`
- `pnpm test`

## Layout
- `apps/editor-web`
- `apps/editor-desktop`
- `apps/marketing-share-web`
- `packages/shared`
- `packages/editor-core`
- `packages/ui`
- `packages/editor-shell`
- `packages/platform`
